# Cerberus Commons

The Commons package provides **shared models, services, and utilities** used across the Cerberus platform.

---

# Purpose

Avoid code duplication between:

- Backend
- Engine
- Blockchain

---

# Contents

- Shared models
- Database access
- Uniswap services
- Crypto utilities
- Blockchain helpers

---

# Tech Stack

- TypeScript
- Prisma Client
- ethers.js

---

# Build

npm run build

---

# Usage

Imported as a shared package inside the monorepo.
