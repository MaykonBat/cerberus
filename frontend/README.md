# Cerberus Frontend

The frontend is a **React application** that provides the user interface for the Cerberus platform.

---

# Features

- MetaMask authentication
- Automation management
- Real-time trade monitoring
- TradingView charts
- Top Uniswap pools
- Top trades dashboard

---

# Environment Variables

Example in the file `.env.example`

---

# Running Frontend

Install dependencies:

- npm install

Start development server:

- npm start

---

# Dashboard

The dashboard provides:

- active trades
- PnL
- automation status
- top pools
- trading charts
